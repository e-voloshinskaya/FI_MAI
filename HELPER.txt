Как собирать программу из нескольких файлов?
1. gcc -c f1.c f2.c ... fn.c. !!! Заголовочные файлы, .h, не включаются в список !!!
2. Будут созданы f1.o ... fn.o
3. gcc -o [название программы] f1.o ... fn.o
4. Получим исполняемый файл [название программы]
5. Запускаем как обычно ./test [параметры]

Как отправить изменения в гитхаб?
1. git status - узнаем, в каком состоянии локальный репозиторий относительно основного, онлайн
2. Если лок. репозиторий устарел, то ??? сначала сделаем копию измененных данных в другом месте
3. git pull, если локальный репозиторий содержит устаревшие данные
4. git add [путь к файлу из тек папки]
5. git commit -m "Комментарий 1"
6. git push и ввести логин-пароль
